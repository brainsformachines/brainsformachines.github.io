## Orange Pi One

![Orange Pi One](https://www.armbian.com/wp-content/uploads/2018/02/orangepione.png)

Orange Pi One is a open-source single board computer. It uses the AllWinner H3 SoC, and has 512MB/1GB DDR3 SDRAM.
